# sobes-backend-restaurant
Репозиторий для реализации тестового задания. Реализация обработки ресторанной логики

## Общее понимание:
1. Буквально делаем для администратора поле "официант", а для официанта - поле "кухня"
1. Условно `while True`
1. Пользователь - админ, который получает отчёты и кидает JSON'ы
1. Клиенты заходят сразу при запуске программы
1. Администратор валидирует заказ на то, что он не пустой и возраст не больше 90. Передаёт "Официанту"
